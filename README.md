# TileMaster DIY

A simple, mobile-friendly web application to help DIYers calculate tiling materials and locate the nearest supply shops.

## How to Use
1. Enter the total area of your project in square meters.
2. Enter the dimensions of your chosen tiles in millimeters.
3. Hit **Calculate** to see your shopping list (includes a 10% wastage buffer).
4. Tap **Find Nearby Tile Shops** to see local suppliers.

## Built With
- HTML5 / CSS3
- Vanilla JavaScript
- Geoapify Places API (for shop location data)

## Setup
To enable the shop finder, get a free API key from [Geoapify](https://www.geoapify.com/) and paste it into the `API_KEY` variable in `index.html`.
